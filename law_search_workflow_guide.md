# 법령 검색 워크플로우 가이드

## 🔍 권장 검색 순서 (개선됨)

### 사용자 질문 유형별 자동 처리

#### 1. "도로교통법의 운전면허 관련 규정 알려줘"
**LLM이 자동으로 처리:**
```
get_law_summary("도로교통법", "운전면허")
```
**내부 동작:**
1. search_law("도로교통법") → MST 확인
2. search_law_articles_semantic(mst="268547", query="운전면허") → 관련 조문 찾기
3. 결과: 제80조(운전면허), 제82조(운전면허시험) 등 관련 조문 표시

#### 2. "공인중개사법 중개수수료 조항 자세히 보여줘"
**LLM이 자동으로 처리:**
```
get_law_summary("공인중개사법", "중개수수료", show_detail=True)
```
**내부 동작:**
1. search_law("공인중개사법") → MST 확인  
2. search_law_articles_semantic(mst="257205", query="중개수수료") → 제32조 찾기
3. get_law_article_by_key(mst="257205", article_key="제32조") → 전체 내용

#### 3. "개인정보보호법 개요"
**LLM이 자동으로 처리:**
```
get_law_summary("개인정보보호법")
```
**내부 동작:**
1. search_law("개인정보보호법") → 기본 정보
2. get_law_detail_unified(mst="248613") → 조문 인덱스

## 🎯 도구별 역할 (새로운 구조)

| 도구 | 사용자 노출 | 용도 | 언제 사용 |
|------|------------|------|----------|
| **get_law_summary** | ✅ 메인 | 법령 요약 + 특정 내용 검색 | 사용자가 법령 내용 질문 시 |
| search_law_unified | ✅ 탐색용 | 범용 법령 탐색 | 법령명을 모를 때 |
| search_law | ⚠️ 제한적 | 정밀 법령명 검색 | 정확한 법령명을 알 때 |
| search_law_articles_semantic | ❌ 내부용 | 조문 내용 검색 | get_law_summary가 자동 호출 |
| get_law_detail_unified | ⚠️ 제한적 | 법령 요약 조회 | 단순 조문 목록만 필요할 때 |
| get_law_article_by_key | ⚠️ 제한적 | 특정 조문 상세 | 조문 번호를 정확히 알 때 |

## 💡 사용 가이드

### ✅ 권장 사용법
```
1. 일반 질문: "도로교통법 운전면허 규정" 
   → get_law_summary("도로교통법", "운전면허")

2. 상세 질문: "공인중개사법 중개수수료 조항 자세히"
   → get_law_summary("공인중개사법", "중개수수료", True)

3. 개요 질문: "개인정보보호법 요약"
   → get_law_summary("개인정보보호법")
```

### ❌ 피해야 할 사용법
```
1. search_law_articles_semantic 직접 호출
   → get_law_summary가 자동으로 처리

2. 복잡한 다단계 검색
   → get_law_summary 하나로 해결
```

## 📊 개선 효과

1. **사용자 편의성**: 한 번의 도구 호출로 모든 정보 제공

3. **정확성**: 시맨틱 검색으로 관련 조문 정확히 찾기
4. **완전성**: 요약부터 상세 내용까지 단계별 제공
