# 한국 법제처 API - 법령 도구 종합 테스트 가이드

> 💡 **이 가이드는 법 지식이 없는 일반인도 쉽게 따라할 수 있도록 작성되었습니다.**
> 
> 각 섹션의 프롬프트를 Claude에게 순서대로 입력하면서 모든 법령 도구를 체험해보세요!

---

## 🏁 1단계: 기본 법령 검색 (일상생활 관련)

### 1-1. 도로교통 관련 법령 찾기
```
자동차 운전할 때 지켜야 할 법령을 검색해주세요. 도로교통법이나 운전 관련 규정이 궁금해요.
```

### 1-2. 검색된 법령의 상세 내용 보기
```
위에서 검색된 도로교통법의 전체 조문을 자세히 보여주세요. 특히 운전면허나 교통사고 처리에 대한 규칙들이 알고 싶어요.
```

### 1-3. 부동산 거래 관련 법령 찾기
```
집을 사거나 전세를 구할 때 알아야 할 법적 규정이 있는지 검색해주세요.
```

### 1-4. 부동산 거래법 상세 내용 확인
```
부동산 거래와 관련된 법령의 전체 내용을 보여주세요. 특히 계약서 작성이나 중개수수료에 대한 부분을 자세히 알고 싶어요.
```
이 테스트를 수행하여 mcp 응답 결과를 분석하고 잘못된 부분이 있는지 보수적으로 평가해줘. 잘못된 부분이 있다면 mcp 도구명과 응답 결과, 잘못된 부분을 알려줘. 보수적으로 평가해줘. 결론이 안나와서 재호출할 필요가 있었다면 그런 부분까지 모두 알려줘.

## 🌍 2단계: 영문 법령 검색 (국제 업무용)

### 2-1. 영문 법령 검색
```
외국인 친구에게 한국의 민법에 대해 설명해야 하는데, 영어로 번역된 한국 법령이 있나요? 'Civil Act'로 검색해주세요.
```

### 2-2. 영문 법령 상세 내용
```
위에서 찾은 영문 민법의 상세 내용을 보여주세요. 외국인이 한국에서 계약할 때 알아야 할 내용들이 궁금해요.
```

### 2-3. 상법 영문 검색
```
한국의 상법도 영어 번역본이 있는지 'Commercial Act'로 검색해주세요.
```

```

## 📚 4단계: 법령 변경 이력 (법이 어떻게 바뀌었는지)

### 4-1. 도로교통법의 변경 이력

```

### 4-2. 특정 연혁의 상세 내용
```

```

### 4-3. 부동산 거래법 변경 이력
```
부동산 거래 관련 법령도 시간이 지나면서 많이 바뀌었을 것 같은데, 변경 이력을 확인해주세요.
```

## 💡 추가 팁

1. **오류 발생 시**: 검색어를 더 간단하게 바꿔보세요 (예: "도로교통법" → "교통")
2. **결과가 없을 때**: 더 일반적인 키워드를 사용해보세요
3. **너무 많은 결과**: 더 구체적인 검색어를 사용하거나 필터를 활용하세요
4. **영문 검색**: 정확한 영문명을 모르면 한글로 먼저 찾은 후 관련 영문법령을 찾아보세요
